# 👤 p1 - backend + database (trọng hoàng)
| Week | Tasks |
|------|------|
| 1 | thiết kế database chi tiết gồm các bảng: users (thông tin user), tests (đề thi), questions (câu hỏi), options (đáp án), attempts (lượt làm bài), xác định khóa chính, khóa ngoại và quan hệ giữa các bảng, setup cấu trúc project PHP (folder api, config, models) |
| 2 | xây dựng API lấy đề thi: viết các endpoint PHP để trả về danh sách đề thi và chi tiết đề (bao gồm câu hỏi và đáp án), format dữ liệu JSON để frontend dễ render |
| 3 | tối ưu database và query: viết JOIN hợp lý giữa các bảng, giảm số lần query, đảm bảo load đề nhanh, không bị dư dữ liệu, kiểm tra lỗi dữ liệu (null, thiếu liên kết) |
| 4 | xây hệ thống phân quyền (RBAC): phân biệt user và admin, kiểm tra quyền truy cập ở backend (ví dụ: chỉ admin mới tạo/sửa đề) |
| 5 | debug và tối ưu: fix bug liên quan API, kiểm tra hiệu năng khi load nhiều câu hỏi, đảm bảo hệ thống ổn định khi demo |
